# agent-wp

Agent WP Plugin

## Dev Setup

### VS Code

- Install PHP Intelephense

### Environment

We provide a Docker configuration for easy setup of a local development environment for the plugin.

### Requirements

- Docker Desktop

### Starting the Environment

Run the following command in the root of the project:

```bash
docker-compose up -d
```
